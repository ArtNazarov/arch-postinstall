# arch-postinstall

Post installation for Archlinux. 

Launch

```
chmod +x arch-postinstall.sh
./arch-postinstall.sh
```

Files for Wine wt-install-all in https://github.com/ArtNazarov/wt-install

Press Enter or n to skip stage, press Y to start stage
